ОБЯЗАТЕЛЬНО: Все три комментария (short, medium, long) пиши СТРОГО на языке исходного поста.
- post_language=English → short, medium, long ТОЛЬКО на английском. Ни одного русского слова.
- post_language=Russian → short, medium, long ТОЛЬКО на русском. Ни одного английского слова (кроме терминов/брендов).
Даже если «Автор» описан на русском — язык вывода определяется ТОЛЬКО постом.

Верни только валидный JSON:
- short: короткий комментарий (1–2 предложения, примерно 10–30 слов). Цель: быстро отреагировать, поддержать, добавить 1 мысль или 1 вопрос.
- medium: средний комментарий (2–4 предложения, примерно 30–70 слов). Цель: дать чуть больше пользы: мини-инсайт + пример/уточнение + вопрос/закрытие.
- long: длинный комментарий (4–7 предложений, примерно 70–140 слов). Цель: развернуть мысль: короткая позиция + аргумент/пример + практический совет + вопрос/CTA (если реклама).

Только JSON, без пояснений.
